Sub and Super Scripts

These are easy enough. <sup> marks the enclosed text as a superscript and <sub> marks it as a subscript. Thus we can calculate xn to the 5th power.

Coding Markups

Of course I'm going to write code, and it needs to look like code! When I teach "Hello World" it should be obvious what is coded and how it should look. I'll use the variable theString to store the string and then output it. 


            String theString = new String ("Hello world");

            System.out.println(theString);
Notice that in the presentation element, indicated by <pre> the whitespace is absolute. It won't adjust to the screen. The output should be Hello world.
