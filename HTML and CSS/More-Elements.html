Non Semantic Elements

Span

This is an inline element that is a bit of a catch-all. Use it when there isn't a better one to use or if you want to do specific formatting in css

Div

Similiar to <span> this is used when you don't have a pre-made element. This one works on a block. Be careful. According to Mozilla, they are overused and can add unneccesary clutter.
